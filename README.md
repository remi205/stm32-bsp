# stm32-bsp

Dans ce repositorie on place les fichiers issues de MicroXplorer propre a ce projet. Ce dépot sera inclus 
dans la configuration tvc-stm32 par un sous module git.

Ainsi ce dépot peut servir d'échange entre une génération ultérieur de MicroXplorer et le projet encore.

Dans l'esprit, ce depot est different a chaque projet mais le fait de placer les fichiers générés dans ce dépot permet de récuperer un ou plusieurs configuration ip.
